# Project 1 - CSV to Azure SQL Database using Azure Data Factory

## Objective
Load employee CSV data from Azure Blob Storage into Azure SQL Database using Azure Data Factory.

## Architecture

CSV File
↓
Azure Blob Storage
↓
Azure Data Factory
↓
Azure SQL Database

## Technologies Used
- Azure Data Factory
- Azure SQL Database
- Azure Blob Storage
- SQL

## Steps Performed
1. Created Azure SQL Database.
2. Created Employees table.
3. Uploaded employees.csv to Storage Account.
4. Created Linked Services.
5. Created Datasets.
6. Created Copy Activity Pipeline.
7. Triggered Pipeline.
8. Verified Data.

## Outcome
Successfully loaded employee data into Azure SQL Database.
